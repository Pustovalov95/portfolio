Здесь я использовал:
-Сетку bootstrap
-Сброс настроек браузера от Bootstrap(для того, чтобы все отображалось так, как нужно в любом браузере)
-Препроцессор SASS(миксины, переменные)
-Flexbox верстку
-Адаптивная верстка под все основные разрешения.
-Шрифты подключал через ссылку на fonts.google.com для удобвства. Могу подключать шрифты через файл css
Всю работу выполнял в VS code.
